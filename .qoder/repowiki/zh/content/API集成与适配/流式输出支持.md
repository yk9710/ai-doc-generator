## 简介

本项目是一个基于React的AI文档生成器，专注于提供流式输出支持的实时AI交互体验。流式输出允许用户在AI模型生成内容的过程中实时看到输出结果，而不是等待整个响应完成后再显示。这种技术显著改善了用户体验，提供了更流畅、更直观的交互体验。

项目支持7个主要的AI提供商（小米MiMo、OpenAI、Anthropic Claude、智谱AI、月之暗面Kimi、DeepSeek、阿里云通义千问），每个提供商都有其独特的API格式和流式输出实现方式。

```mermaid
sequenceDiagram

Reader->>Parser : 读取数据块
Parser->>Parser : 解析SSE事件
Parser->>UI : onChunk(增量内容)
UI->>UI : 实时更新显示
Parser->>Parser : 检测[DONE]结束
Parser->>UI : onComplete()
UI->>UI : 完成渲染
```

### 流式数据处理算法

流式输出的核心处理逻辑如下：

```mermaid
flowchart TD
Start([开始流式处理]) --> InitBuffer[初始化缓冲区]
InitBuffer --> ReadLoop[进入读取循环]
ReadLoop --> ReadData[读取数据块]
ReadData --> CheckDone{是否完成?}
CheckDone --> |是| Complete[触发完成回调]
CheckDone --> |否| DecodeData[解码数据]
DecodeData --> SplitLines[按行分割]
SplitLines --> ProcessLines[处理每行数据]
ProcessLines --> CheckSSE{是否SSE行?}
CheckSSE --> |是| ParseJSON[解析JSON数据]
CheckSSE --> |否| ReadLoop
ParseJSON --> ExtractContent[提取内容片段]
ExtractContent --> CheckContent{内容存在?}
CheckContent --> |是| CallChunk[调用onChunk回调]
CheckContent --> |否| ReadLoop
CallChunk --> ReadLoop
Complete --> End([结束])
```
