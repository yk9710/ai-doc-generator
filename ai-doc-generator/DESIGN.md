# 🎨 界面设计说明

## 设计理念

**赛博朋克科技美学** - 融合未来主义与赛博朋克风格，打造沉浸式的AI创作体验。

### 核心设计原则

1. **霓虹光效** - 青色（Cyan）和品红色（Magenta）为主色调，营造科技感
2. **网格背景** - 动态网格动画，增强空间深度
3. **发光效果** - 边框发光、阴影发光，营造霓虹氛围
4. **字体层次** - Orbitron（标题）+ Rajdhani（正文）+ JetBrains Mono（代码）
5. **动态交互** - 悬停效果、入场动画、脉冲动画

### 动效设计

#### 1. 头部辉光动画
```css
@keyframes headerGlow {
  0% { box-shadow: 0 0 20px rgba(0, 245, 255, 0.3); }
  100% { box-shadow: 0 0 40px rgba(255, 0, 255, 0.4); }
}
```
头部在青色和品红色之间渐变发光。

#### 2. 边框旋转动画
```css
@keyframes borderRotate {
  0% { filter: blur(10px) hue-rotate(0deg); }
  100% { filter: blur(10px) hue-rotate(360deg); }
}
```
头部边框颜色循环变化。

#### 3. 扫描线效果
```css
@keyframes scanLine {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}
```
标题下方的光线扫描效果。

#### 4. 网格背景移动
```css
@keyframes gridMove {
  0% { transform: translate(0, 0); }
  100% { transform: translate(50px, 50px); }
}
```
背景网格缓慢移动，营造空间感。

#### 5. 消息入场动画
```css
@keyframes slideIn {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
```
每条消息从下方滑入，延迟递增。

#### 6. 机器人脉冲动画
```css
@keyframes pulse {
  0%, 100% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.1); opacity: 0.8; }
}
```
空状态机器人图标呼吸效果。

## 交互设计

### 悬停效果

#### 配置面板
- 悬停时上浮 5px
- 发光效果增强
- 过渡时间 0.3s

#### 模板按钮
- 悬停时右移 5px
- 光线扫过动画
- 边框发光

#### 发送按钮
- 悬停时上浮 3px + 放大 1.02
- 阴影增强
- 点击水波纹效果

### 焦点状态

#### 输入框
- 聚焦时边框变为青色
- 添加青色发光
- 背景色加深

### 按钮状态

| 状态 | 样式 |
|------|------|
| 默认 | 渐变背景 + 阴影 |
| 悬停 | 上浮 + 阴影增强 |
| 禁用 | 透明度 0.5 + 不可点击 |
| 点击 | 水波纹扩散 |

## 组件设计

### 1. 头部区域
- **背景**: 半透明渐变 + 模糊效果
- **边框**: 发光边框 + 旋转颜色动画
- **标题**: Orbitron 字体 + 渐变色 + 扫描线
- **描述**: Rajdhani 字体 + 字母间距

### 2. 配置面板（左侧）
- **定位**: 粘性定位，滚动时固定
- **背景**: 半透明深色 + 模糊
- **边框**: 青色发光
- **悬停**: 上浮 + 发光增强

### 3. 对话面板（右侧）
- **背景**: 半透明深色 + 模糊
- **边框**: 品红色发光
- **消息**: 
  - 用户消息：青色边框
  - AI消息：品红色边框
  - 入场动画：滑入效果

### 4. 输入区域
- **文本框**: 
  - 半透明背景
  - 聚焦时青色发光
  - 平滑过渡
- **按钮组**:
  - 发送：渐变背景
  - 清空：红色半透明
  - 导出：绿色半透明

## 响应式设计

### 断点

| 断点 | 布局 |
|------|------|
| > 1200px | 双栏布局（400px + 自适应） |
| 768px - 1200px | 单栏布局 |
| < 768px | 紧凑布局，缩小字体和间距 |

### 移动端优化
- 配置面板取消粘性定位
- 减小字体大小
- 减小内边距
- 优化触摸区域

## 性能优化

### CSS 优化
1. 使用 CSS 变量统一管理
2. 使用 transform 代替 position 动画
3. 使用 will-change 提示浏览器
4. 减少重绘和重排

### 动画优化
1. 使用 CSS 动画代替 JS 动画
2. 使用 requestAnimationFrame
3. 减少动画复杂度
4. 适当使用 will-change
