## 二、项目详细描述

### 2.1 项目概述
AI 文档生成器是一款**多模型AI应用平台**，专注于为开发者和内容创作者提供高效的文档生成、代码辅助和技术写作服务。应用采用现代化的 React + Vite 技术栈，支持 7 种主流 AI 提供商（MiMo、OpenAI、Claude、智谱、Kimi、DeepSeek、通义千问），提供直观的可视化界面和流畅的交互体验。

**核心设计理念**：不依赖单一模型，为用户提供灵活多样的AI选择。

### 2.2 核心功能

#### 📝 智能文档生成
- **技术文档**: 自动生成包含概述、架构设计、API 接口、使用示例的完整技术文档
- **API 文档**: 为接口生成详细的请求参数、响应格式和示例代码
- **教程指南**: 创建从入门到进阶的完整教程，包含实践案例

#### 💻 代码辅助
- **代码生成**: 根据需求描述生成规范、有注释、包含错误处理的代码实现
- **代码审查**: 自动审查代码并提供优化建议
- **代码解释**: 解释复杂代码逻辑，帮助理解和学习

#### 💬 对话交互
- **多轮对话**: 支持上下文记忆的多轮对话交互
- **模板系统**: 预置多种文档模板，快速开始创作
- **自定义提示词**: 灵活的自定义提示词功能，满足个性化需求

### 2.3 技术架构

#### 前端技术栈
- **框架**: React 19 + Vite 5
- **Markdown 渲染**: react-markdown
- **HTTP 客户端**: Axios
- **样式**: 原生 CSS + 渐变设计

#### MiMo API 集成（主要）
- **模型版本**: MiMo V2.5
- **调用方式**: RESTful API (支持流式输出)
- **应用场景**: 
  - 文档生成 (文本推理模型)
  - 代码辅助 (代码推理模型)
  - 多模态理解 (未来扩展)

#### 多模型架构
- **支持的提供商**: 7种 (MiMo, OpenAI, Claude, 智谱, Kimi, DeepSeek, Qwen)
- **统一接口**: 抽象化API调用层，支持OpenAI兼容格式
- **灵活切换**: 用户可根据需求选择最合适的模型
- **对比测试**: 支持不同模型输出质量对比

#### 关键特性
- ✅ 完整的错误处理机制
- ✅ API Key 验证和管理
- ✅ 响应式界面设计
- ✅ 实时加载状态反馈
- ✅ Markdown 格式渲染

## 七、技术对接方案

### 7.1 多模型 API 集成方式

**统一调用接口**：
```javascript
// 支持多种 AI 提供商的统一调用
const response = await callAIProvider({
  provider: 'mimo',  // mimo|openai|anthropic|zhipu|moonshot|deepseek|qwen
  apiKey: 'your-api-key',
  model: 'mimo-v2.5',
  prompt: '用户输入',
  history: []  // 对话历史
})
```

**OpenAI 兼容格式**：
```javascript
// 标准 OpenAI 格式，MiMo 和其他提供商均支持
const response = await axios.post(
  'https://api.mimo.xiaomimimo.com/v1/chat/completions',
  {
    model: 'mimo-v2.5',
    messages: [
      { role: 'system', content: '系统提示词' },
      { role: 'user', content: '用户输入' }
    ],
    temperature: 0.7,
    max_tokens: 4000
  },
  {
    headers: {
      'Authorization': `Bearer ${apiKey}`
    }
  }
)
```

**多提供商配置**：
```javascript
export const PROVIDERS = {
  mimo: {
    name: '小米 MiMo',
    apiUrl: 'https://api.mimo.xiaomimimo.com/v1/chat/completions',
    models: ['mimo-v2.5', 'mimo-v2.5-lite', 'mimo-vision']
  },
  openai: {
    name: 'OpenAI',
    apiUrl: 'https://api.openai.com/v1/chat/completions',
    models: ['gpt-4o', 'gpt-4o-mini', 'gpt-4-turbo']
  },
  // ... 更多提供商
}
```
